- 0、**安装依赖：** `npm i`
- 1、**启动：** `npm run dev` 开启调试服务
- 2、**开发：** 编辑 `/src/index.html` 文件，Web服务实时预览
- 3、**生产：** `npm run build` 打包构建
- 4、**预览：** 浏览器打开 `/index.html` 查看结果

> chrome浏览器 > 打印 > 目标另存为`PDF` > 更多设置无边距 > 即可导出
